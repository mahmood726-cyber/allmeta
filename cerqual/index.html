GRADE-CERQual

Confidence in Evidence from Reviews of Qualitative research (Lewin et al., PLOS Med 2018). Rate four components per review finding: methodological limitations, coherence, adequacy of data, and relevance. The tool derives an overall confidence rating from the component concerns.

Summary heatmap

Default rating uses worst-component anchoring — no numeric thresholds. All "No" or "Very minor" → High; Moderate in any component → Moderate; Serious in any → Low (or Very low if serious in ≥2 components). This reproduces the qualitative reasoning in Lewin 2018; reviewers should justify any manual override narratively.
