Which kind of review are you running?

Each workflow is a curated sequence of allmeta tools in methodological order — from question framing through synthesis and reporting. Pick the one that matches your evidence question. You can always combine tools later directly from the catalogue.

Systematic review

The full PRISMA 2020 sequence: PICO → search → dedupe → dual screen → RoB → synthesis → GRADE.

When you have time and want Cochrane-level rigour

Rapid review

Cochrane Rapid Reviews Methods Group (Garritty 2021) — single-reviewer screening with verification, streamlined extraction, primary-outcome RoB.

When you have a deadline (decision in days/weeks)

Prognostic review

PROGRESS framework + PICOTS, CHARMS extraction, QUIPS / PROBAST, GRADE-for-prognosis (Foroutan 2020).

When the question is about prognostic factors or prediction models

Network meta-analysis

Multi-treatment comparisons: transitivity, consistency, node-splitting, SUCRA ranking, CINeMA confidence.

When the question compares more than two interventions

Diagnostic test accuracy

QUADAS-2 appraisal, bivariate / HSROC pooling, threshold-effect detection, GRADE-DTA.

When the outcome is a diagnostic test's sensitivity / specificity

Not sure which one fits? Most clinical-effectiveness reviews start with systematic review. For a guideline urgency, see rapid review. Each workflow page links the same allmeta tools — picking the wrong one isn't a hard error.
