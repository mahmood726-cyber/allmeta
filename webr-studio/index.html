WebR (webr.r-wasm.org) is a full R distribution compiled to WebAssembly. First load fetches ~30 MB; packages are installed from the webr binary repo on first use. Everything runs locally — your data never leaves the browser. For very large models or real MCMC (Stan), see the Local R Install recipe.
