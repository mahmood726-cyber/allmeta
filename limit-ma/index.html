Limit meta-analysis (Rücker)

Adjusts the pooled effect for small-study effects by extrapolating to the zero-variance limit via a shrunken random-effects model. Rücker et al., Stat Med 2011.

Input

Pooled vs limit

MethodEstimateSE95% CI

Funnel + Rücker regression

Interpretation: the limit estimate is the value the random-effects model would return if all studies had zero sampling error. A large gap between the standard random-effects estimate and the limit estimate indicates small-study effects (possibly pub bias).
Rücker G, Schwarzer G, Carpenter JR, Binder H, Schumacher M. Treatment-effect estimates adjusted for small-study effects via a limit meta-analysis. Biostatistics 2011;12:122-142. Browser implementation: shrunken RE fit then linear extrapolation of predicted effect at variance = 0.
