Component NMA

Additive model for multi-component interventions (Welton et al., Am J Epidemiol 2009). Each intervention is a set of components (e.g., "drug + exercise + education"). The model estimates one effect per component; any combination's effect is the sum of its components.

Assumption: component effects are additive on the outcome scale you used. If interactions between components matter (synergy / antagonism), add interaction terms (e.g., include drug&exercise as a separate component) or use a full interaction model.
Welton NJ, Caldwell DM, Adamopoulos E, Vedhara K. Mixed treatment comparison meta-analysis of complex interventions: psychological interventions in coronary heart disease. Am J Epidemiol 2009;169:1158-1165.
