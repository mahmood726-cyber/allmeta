CINeMA — Confidence in NMA

Rate each pairwise comparison in a network meta-analysis across six CINeMA domains (Nikolakopoulou et al., PLOS Med 2020): within-study bias, reporting bias, indirectness, imprecision, heterogeneity, incoherence. The tool derives a confidence rating from domain downgrades.

Rating rule used: each domain contributes 0 (No concerns), 1 (Some concerns), or 2 (Major concerns) downgrade points. Total 0 → High; 1-2 → Moderate; 3-4 → Low; ≥5 → Very low. Reviewers may override based on judgment; the tool is an aid, not a substitute.
