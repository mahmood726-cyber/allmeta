PowerMA / Required Information Size

Plan a meta-analysis: given a target effect size, α, β, and anticipated heterogeneity, what total information (N) is needed to detect that effect? Includes RIS calculation per Trial Sequential Analysis (TSA) conventions.

Power curve (N across possible true effects)

Formula (continuous): N_per_arm = 2·(z_{1-α/2} + z_{1-β})² / (MD/SD)². For binary with RRR: effect size d = arcsin√p1 − arcsin√p0 and N_per_arm = (z_{1-α/2} + z_{1-β})² / (d²·fractional allocation). RIS = N_per_arm × 2 × (1/(1 − D²)) when D² is the diversity inflation (Wetterslev 2008).
