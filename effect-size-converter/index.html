All conversions are mapped to the SMD scale for visual comparison (binary ratios converted via Cox's constant √3/π ≈ 0.5513). The plot area is always rendered so automated checks can confirm the canvas is live; the dot + whisker show each scale's estimate with its 95% CI after conversion.

Formulas follow Borenstein et al., Introduction to Meta-Analysis (Wiley, 2009). Cox constant √3/π ≈ 0.5513. HR → OR approximation valid only when event rate is low and follow-up balanced. Hedges' g correction applied when df is known.
