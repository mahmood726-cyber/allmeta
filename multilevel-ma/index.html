Multi-level (3-level) Meta-Analysis

When each study contributes multiple effect sizes (e.g., several outcomes, sub-scales, or time points), ignoring that dependency underestimates SEs. The 3-level model adds a study-level random effect τ²_study on top of the effect-level τ²_within.

Input

Pooled estimate

Effect sizes

StudyLabelEffectSEWeight

Forest (grouped by study)

Variance decomposition: var(θ_ij) = σ²_ij (within-effect sampling) + τ²_within (within-study heterogeneity across effect sizes) + τ²_study (between-study). Estimation: iterated REML-style method-of-moments (Cheung 2014 simplification of metafor::rma.mv). For robust inference with highly clustered data, RVE is the gold standard.
